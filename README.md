```

R8, R9: 27K
```

```
IC1: 74LS157

IC3,IC4: 75176 montés sur support (important, doivent être remplaçables)
IC5: 74LS123
IC6: 74LS393
IC7: 74LS14
IC8: 74LS04
IC9: 74LS11
IC10,IC11: 74LS138
IC12: 74LS139
IC13: 74LS245 monté sur support
IC14: D7201C (Nec) monté sur support
IC15: D8253C-5 (Nec) ou 8253-5 (AMD) monté sur support
```
